1.1.12.1.4.14 DRV_AT25DF_TRANSFER_STATUS Enum

C

typedef enum
{
    /* Transfer is being processed */
    DRV_AT25DF_TRANSFER_STATUS_BUSY,

    /* Transfer is successfully completed*/
    DRV_AT25DF_TRANSFER_STATUS_COMPLETED,

    /* Transfer had error */
    DRV_AT25DF_TRANSFER_STATUS_ERROR

} DRV_AT25DF_TRANSFER_STATUS;

Summary

Defines the data type for AT25DF Driver transfer status.

Description

This will be used to indicate the current transfer status of the AT25DF FLASH driver operations.

Remarks

None.
