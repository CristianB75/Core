Running the Application

MCU Devices

  1. Connect a micro USB cable to the DEBUG USB port.
  2. Open the Project basic_freertos_trustZone (Project Group) in the MPLAB X IDE and Set Non-secure project sam_l11_xpro_NonSecure.X as Main Project.
  3. Build and program the application (sam_l11_xpro_NonSecure.X project) using the MPLAB X IDE.
  4. The LED indicates the success or failure.
    • The LED toggles on success i.e. each time when the Task2 or Task3 receives a valid message

 

The following table provides the LED name:
