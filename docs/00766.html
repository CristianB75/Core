Running The Application
  1. Connect a micro USB cable to the DEBUG port.
  2. Build and Program the application using the MPLAB X IDE.
  3. Insert the SD Card in the SD Card slot of the Device.
  4. The LED is turned ON when
    1. The content from the NVM media (HELLO WORLD !) is copied to SD Card successfully
    2. Application has successfully written the last modified time stamp to SD Card

 

The following table provides the LED name:

Kit Name 
LED Name 
SAM E54 Xplained Pro Evaluation Kit 
LED 0 

 

  1. If Success, Insert the SD Card on to your host PC
  2. FILE.txt should have the content

HELLO WORLD ! 

File Modified on [dd/mm/yy hh:mm:ss] 

Note:
  1. The time stamp mentioned above is the compile time of app.c file
  2. If step 3 is repeated then new timestamp is appended to the existing FILE.txt
