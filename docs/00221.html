Hardware Setup
  1. Project pic32mk_gp_db.X.
    • Hardware Used:
      • PIC32MK General Purpose Development Kit
    • Hardware Setup:
      • Use jumper wire to Connect MOSI and MISO lines of J1 connector
        • SPI6 MOSI signal is mapped to RB0
        • SPI6 MISO signal is mapped to RG9
        • Connect a micro USB cable to the DEBUG port.

 

  1. Project pic32mk_mcj_curiosity_pro.X
    • Hardware Used
      • PIC32MK MCJ Curiosity Pro
    • Hardware Setup
      • Use jumper wire to connect "Pin 16 of EXT1 header" to "Pin 17 of EXT1 header"
        • SDI2 is mapped to RD6 that is routed to "Pin 17 of EXT1 header"
        • SDO2 is mapped to RC7 that is routed to "Pin 16 of EXT1 header"
      • Connect micro USB cable to the 'Debug USB' connector(J1) on the board to the computer.

 

  1. Project pic32mx_xlp_sk.X
    • Hardware Used
      • PIC32MX XLP Starter Kit
    • Hardware Setup
      • Connect micro USB cable to the 'Debug USB' connector(J9) on the board to the computer.
      • Use jumper wire to connect "Pin 5 of J5" to "Pin 6 of J5"
        • SDI1 is mapped to RC8 that is routed to "Pin 5 of J5"
        • SDO1 is mapped to RA8 that is routed to "Pin 6 of J5"
        • Connect a micro USB cable to the USB DEBUG port J19

 

  1. Project pic32mz_dag_sk.X.

19. Project sam_a5d2_xult.IAR OR sam_a5d2_xult_xc32.X 

20. Project sam_a5d2_xult_freertos.IAR OR sam_a5d2_xult_xc32_freertos.X

  • Hardware Used
    • SAM A5D2 Xplained Ultra board
    • SD Card with FAT32 file system
  • Hardware Setup
    • Setup the SD card
      • Download harmony MPU bootstrap loader from this location
      • Copy the downloaded boot loader binary( boot.bin) and generated application binary (harmony.bin)into the SD card
      • Insert the SD card into the SDMMC1 slot on the board
    • Use jumper wire to Connect "Pin 16 of EXT1 header" to "Pin 17 of EXT1 header"
      • SPI1 MOSI signal is mapped to PD26 that is routed to "Pin 16 of EXT1 header"
      • SPI1 MISO signal is mapped to PD27 that is routed to "Pin 17 of EXT1 header"
    • Connect the Debug USB port (J14) on the board to the computer using a micro USB cable

 

21. Project sam_9x60_ek.IAR OR sam_9x60_ek_xc32.X 

22. Project sam_9x60_ek_freertos.IAR OR sam_9x60_ek_freertos_xc32.X

  • Hardware Used
    • SAM 9X60 Evaluation Kit board (Rev. B)
    • SD Card with FAT32 file system
  • Hardware Setup
    • Setup the SD card
      • Download harmony MPU bootstrap loader from this location
      • Copy the downloaded boot loader binary( boot.bin) and generated application binary (harmony.bin)into the SD card
      • Insert the SD card into the SDMMC connector J4 on the board
    • Use jumper wire to Connect "Pin 19 of J16 Connector" to "Pin 21 of J16 Connector"
      • FLEXCOM4 MOSI signal is mapped to PA12 that is routed to "Pin 19 of J16 Connector"
      • FLEXCOM4 MISO signal is mapped to PA11 that is routed to "Pin 21 of J16 Connector"
    • Connect the USB port J22 on board to the computer using a micro USB cable (to enable debug com port)
    • Connect the USB port J7 on board to the computer using a micro USB cable (to power the board)
    • Note: Reset push button on SAM9X60 EK is labeled as SW3
